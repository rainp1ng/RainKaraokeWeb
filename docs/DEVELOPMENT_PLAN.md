# RainKaraoke Web 开发方案与排期计划

## 一、开发环境准备

### 1.1 开发工具

| 工具 | 版本 | 用途 |
|------|------|------|
| Node.js | 18+ | JavaScript运行环境 |
| pnpm/yarn/npm | 最新 | 包管理器 |
| VS Code | 最新 | IDE |
| Chrome DevTools | 最新 | 调试工具 |

### 1.2 项目初始化

```bash
# 创建Vite项目
pnpm create vite RainKaraokeWeb --template react-ts

# 安装依赖
cd RainKaraokeWeb
pnpm install

# 安装核心依赖
pnpm add zustand idb react-router-dom
pnpm add -D tailwindcss postcss autoprefixer
pnpm add -D @types/node
```
