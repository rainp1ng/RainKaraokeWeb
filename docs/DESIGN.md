## 二、核心模块设计

### 2.1 播放器模块

```
┌─────────────────────────────────────────────────────────────┐
│                     Player Module                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                  VideoPlayer                         │    │
│  │  ┌────────────────────────────────────────────────┐ │    │
│  │  │ <video> Element                                │ │    │
│  │  │  ├── Video Track                               │ │    │
│  │  │  └── Audio Track                               │ │    │
│  │  └────────────────────────────────────────────────┘ │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                 AudioContext                         │    │
│  │                                                      │    │
│  │  [Source] → [Analyser] → [Gain] → [Destination]     │    │
│  │                ↓                                     │    │
│  │           [Effect Chain]                             │    │
│  │           ├── Reverb                                 │    │
│  │           ├── EQ                                     │    │
│  │           └── Compressor                             │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                              │
│  状态:                                                       │
│  • isPlaying: boolean                                       │
│  • currentTime: number                                      │
│  • duration: number                                         │
│  • isVocal: boolean (原唱/伴唱)                             │

│  • speed: number (0.5 ~ 2.0)                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 五、性能优化策略

### 5.1 加载优化

- 代码分割（路由级别）
- 懒加载组件
- 预加载关键资源
- 图片/音频压缩

### 5.2 运行时优化

- Web Worker处理歌词解析
- requestAnimationFrame更新UI
- 虚拟列表渲染大量歌曲
- 音频缓冲预加载

### 5.3 移动端优化

- 触摸手势支持
- 防止误触
- 适配安全区域
- 减少重排重绘
